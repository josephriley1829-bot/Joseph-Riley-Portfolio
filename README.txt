JOSEPH RILEY ARCHITECTURE PORTFOLIO

HOW TO OPEN:
1. Unzip the folder.
2. Open the folder in VS Codium.
3. Open index.html.
4. Right click index.html and choose "Open with Live Server" if you have the Live Server extension.
   OR double click index.html to open it in Chrome.

FILES:
- index.html = website text, project names, email, and page structure
- style.css = colors, layout, sizing, animation styling
- script.js = scroll animation and cursor effect
- assets folder = all portfolio images

EASY EDITS:
- Change your email near the bottom of index.html: your-email@example.com
- Change project descriptions in each project section.
- Replace images by putting new images in the assets folder and updating the image file name in index.html.
